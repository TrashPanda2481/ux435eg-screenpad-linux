# ux435eg-screenpad-linux

A userspace tool for driving the secondary ScreenPad display on the ASUS ZenBook UX435EG under Linux. Goal: get the thing usable (backlight, basic output) without booting into Windows.

Work in progress. Expect this README to be rewritten once I actually have something working.

## Status

Very early WIP. Currently just poking at USB HID traffic and taking notes. No usable binary yet.

Tested (so far) on:

- ASUS UX435EG, stock firmware
- Debian 12, kernel 6.1

If you have the same laptop and want to help capture traces, open an issue.
